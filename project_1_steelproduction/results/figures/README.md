# Figures

This folder contains all plots generated by the notebooks:

- `actual_vs_predicted.png` — scatter plot comparing true vs. predicted output values for the **Random Forest** model (v1), including a residual plot. Generated by `steelproduction.ipynb`.
- `feature_importance.png` — chart showing which of the input features had the most influence on the Random Forest predictions.
- `actual_vs_predicted_v2.png` — same scatter and residual plot for the **XGBoost** model (v2). Generated by `steelproduction_v2.ipynb`.
- `feature_importance_v2.png` — feature importance chart for the XGBoost model.
