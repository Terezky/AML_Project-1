# Processed Data

This folder contains the normalized datasets used for model training and evaluation:

- `normalized_train_data.csv` —  used to train the model. Contains 22 columns: one `output` column  and 21 input columns (`input1`–`input21`).
- `normalized_test_data.csv` — same data format, used to evaluate and test the model.

All values are already normalized (range [0, 1]).
